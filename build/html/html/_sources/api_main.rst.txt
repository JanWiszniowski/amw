Main module
###########

The main module, ``spectral_mw.py``, contains functions that perform all the operations needed to count Mw magnitudes.
These include finding peaks, taking waveforms, counting spectra, etc.
Mw main module functions are presented below, following the alphabetic order.
The module ``spectral_mw.py`` can be called directly with the configuration file in JsonFormat
and the catalogue file in the QuakeML XML format as parameters

E.g: ``python3 spectral_mw.py config.json catalog.xml``
